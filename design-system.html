Design Tokens & Components

คู่มือระบบการออกแบบสำหรับเว็บไซต์อสังหาริมทรัพย์ (Material Design 3 Based)

Understanding Design Tokens (M3)

Design Tokens คือหน่วยย่อยที่สุดของสไตล์ (Style values) เช่น สี, ตัวอักษร, และระยะห่าง ที่ถูกตั้งชื่อเพื่อให้เข้าใจง่ายและนำไปใช้ซ้ำได้ การใช้ Tokens แทนค่า Hard-coded (เช่น Hex codes) ช่วยให้ระบบมีความยืดหยุ่นและดูแลรักษาง่าย

1. Reference Tokens

ค่าอ้างอิงพื้นฐาน

ค่าดิบที่ยังไม่มีความหมายเชิงระบบ

navy.900
#0F172A
green.500
#4CAF50
2. System Tokens

ค่าเชิงระบบ (Semantic)

การกำหนดบทบาทให้ค่าอ้างอิง *ใช้ระดับนี้เป็นหลัก

primary= navy.900
secondary= green.500
3. Component Tokens

ค่าเฉพาะองค์ประกอบ

นำ System Tokens ไปใช้ในชิ้นงานจริง

bg: primary
